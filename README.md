# YE.LIKESYOU.ORG
---
A minimalist, YEEZY/DONDA-inspired ye music archive. try out [here](https://ye.likesyou.org)

## Features
- automatically scans the /audio subfolder for mp3 files via php
- matches filename.mp3 with filename-cover.png automatically.

## Selfhost
1. Place the index.php file on a server with php support.
2. create the audio folder in the root directory.
3. upload your .mp3 files into the audio folder.
4. (Optional) add cover art by naming images songname-cover.png.

## Deploy (InfinityFree)
1. open [infinityfree.com](https://infinityfree.com)
2. create a new subdomain
3. upload all the files
